BitcoinityBrowser
==============

Is a realy simple Browser App for http://bitcoinity.org/markets

	- No Statusbar
	- Landscape Mode
	- Turn Screen NO Off
	- Side reload
	- Share Sidelink with Friends
	- OpenSource and Free
	- min. 4,8"/10,2cm Display (optimized for Samsung Galaxy S3 and Tablets)
    
In Future:
	Start values(currency,market) coming soon
